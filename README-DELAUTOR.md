Sistema desarrollado en Java utilizando arquitectura en capas (MVC + DAO + Service) para la gestión de reclamos, alegaciones y usuarios.

## Características

- Gestión de usuarios
- Creación de reclamos
- Gestión de alegaciones
- Conexión a base de datos con JDBC
- Arquitectura profesional por capas:
  - Model
  - DAO
  - Controller
  - Service
  - View (Swing)

## Tecnologías utilizadas

- Java
- Maven
- JDBC
- MySQL
- Swing
- Git

## Arquitectura

El sistema implementa:

- Patrón DAO
- Separación de responsabilidades
- Arquitectura en capas
